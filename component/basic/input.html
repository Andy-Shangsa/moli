    Input 输入框

    通过鼠标或键盘输入字符

    基础用法

    <ml-input v-model="model" placeholder="请输入内容"></ml-input>
    显示代码

    禁用状态

    通过 disabled 属性指定是否禁用 input 组件
    <ml-input v-model="model" placeholder="请输入内容" disabled></ml-input>
    显示代码

    可清空

    使用 clearable 属性即可得到一个可清空的输入框
    <ml-input 
v-model="model" 
placeholder="请输入内容" 
clearable>
</ml-input>
    显示代码

    密码框

    使用 show-password 属性即可得到一个可切换显示隐藏的密码框
    <ml-input type="password" v-model="model" show-password></ml-input>
    显示代码

    文本域

    用于输入多行文本信息，通过将 type 属性的值指定为 textarea。

    文本域高度可通过 rows 属性控制
    <ml-input 
type="textarea" 
v-model="textarea" 
placeholder="请输入内容"  
:rows="2">
</ml-input>
    显示代码

    可自适应文本高度的文本域

    通过设置 autosize 属性可以使得文本域的高度能够根据文本内容自动进行调整，并且 autosize 还可以设定为一个对象，指定最小行数和最大行数。

    <ml-input
type="textarea"
:autosize="{ minRows: 2, maxRows: 4}"
placeholder="请输入内容"
v-model="textarea2">
</ml-input>
    显示代码

    type 类型 string text，textarea 和其他 原生 input 的 type 值 text
